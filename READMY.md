# Dostaffkin — Калькулятор доставки с Яндекс Картами

Веб-приложение для расчёта стоимости доставки с визуализацией маршрута на картах. Пользователь выбирает адреса, размер груза, скорость доставки и получает итоговую цену.

## Основной функционал

- **Интерактивная карта** — отображение маршрута через Яндекс.Карты
- **Подсказки адресов** — автодополнение при вводе городов/улиц
- **Расчёт стоимости** — по формуле: тариф × расстояние (км) + минимальная цена размера
- **Выбор размера посылки** — 6 вариантов (XS → MAX)
- **Приоритетная доставка** — +15% к цене, -30% к сроку
- **Оформление заявки** — отправка данных на бэкенд

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Фреймворк | Angular (standalone components, сигналы) |
| Карты | Яндекс.Карты API 2.1 |
| Формы | ReactiveFormsModule |
| HTTP-клиент | @angular/common/http |
| Стили | Vanilla CSS |
